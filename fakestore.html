Fake Store

The assignment is to create a one-page application using React framework. I choose to create a online shopping store from https://fakestoreapi.com/. In this shopping website, users are able to add items in to the cart and see the total amount and price. They can also checkout. To see only items in some categories or in some particular order, users can filter by category and price range and sort items by title or price.

Usability Principles

This website has three major components: navigation bar, cart, and product grid. With these components, I incoporate the following design principles in the website.

  1. Minimalist design — Only relevant information and components are included to prevent confusion from the novice user.
  2. Flexibility — A user can filter and sort items shown in the product grid.
  3. Help users recognize and recover from errors — The amount of each item in the cart is displayed to prevent users from buying incorrect amount. To recover from errors, users can add one unit, remove one unit, and delete all item units from the cart.
  4. Error prevention — When there is only one unit of item left in the cart, the user cannot remove the item by clicking minus sign. They must click Delete. This is to prevent removing too many item units from the cart.
  5. Visibility of system status — When users apply two filters yet no items are in both categories, the following text will be shown: No items found in the selected category and price range. Similarly, when the website is fetching API data, Loading will be shown.
  6. Consistency and standards — All card of sale item have the same pattern.

Reflection

Since this assignment limits the website to be in one page, there are some limitations, such as cart and confirmation cannot be in a different pages. In other words, this assignment focuses on React usage and less on UI/UX design. Nevertheless, I included my design philosophy — simplicity, comprehension, inclusivity, and functionality — in the assignment as usual.
